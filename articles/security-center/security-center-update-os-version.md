# <a name="update-os-version-in-azure-security-center"></a>Zaktualizuj wersję systemu operacyjnego w Centrum zabezpieczeń Azure
Dla maszyn wirtualnych (VM) usług w chmurze Centrum zabezpieczeń Azure zaleca czy systemu operacyjnego (OS) można zaktualizować, jeśli jest dostępna nowsza wersja.  Tylko chmury usługi sieci web i proces roboczy role uruchomione w środowisku produkcyjnym, które są monitorowane gniazda.

> [!NOTE]
> Informacje na temat usługi przedstawiono w tym dokumencie za pomocą przykładowego wdrożenia.  Nie jest to przewodnik krok po kroku.
> 
> 

## <a name="implement-the-recommendation"></a>Wykonania zalecenia
1. W **zalecenia** bloku, wybierz opcję **wersji systemu operacyjnego aktualizacji**.
   ![Aktualizacja wersji systemu operacyjnego][1]
2. Spowoduje to otwarcie bloku **wersji systemu operacyjnego aktualizacji**. Wykonaj kroki opisane w tym bloku, aby zaktualizować wersję systemu operacyjnego.
