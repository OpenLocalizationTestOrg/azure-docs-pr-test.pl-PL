<span data-ttu-id="edb56-104">Centrum zabezpieczeń Azure zaleca zastosowanie szyfrowania dysków, jeśli masz dysków systemu Windows lub maszyny Wirtualnej systemu Linux, które nie są szyfrowane przy użyciu szyfrowania dysków Azure.</span><span class="sxs-lookup"><span data-stu-id="edb56-104">Azure Security Center recommends that you apply disk encryption if you have Windows or Linux VM disks that are not encrypted using Azure Disk Encryption.</span></span> <span data-ttu-id="edb56-105">Szyfrowanie dysku umożliwia zaszyfrowanie dysków systemu Windows i maszyny Wirtualnej systemu Linux IaaS.</span><span class="sxs-lookup"><span data-stu-id="edb56-105">Disk Encryption lets you encrypt your Windows and Linux IaaS VM disks.</span></span>  <span data-ttu-id="edb56-106">Szyfrowanie jest zalecane zarówno w przypadku woluminów z systemem operacyjnym, jak i z danymi maszyny wirtualnej.</span><span class="sxs-lookup"><span data-stu-id="edb56-106">Encryption is recommended for both the OS and data volumes on your VM.</span></span>
