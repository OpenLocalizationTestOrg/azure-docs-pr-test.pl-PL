# <a name="azure-storage-samples"></a>Przykłady usługi Azure Storage

Skorzystaj z poniższych łączy, aby wyświetlić i pobrać usługi Azure Storage przykładowy kod i aplikacji.

## <a name="azure-code-samples-library"></a>Biblioteka przykłady kodu platformy Azure
[Przykłady kodu Azure](https://azure.microsoft.com/documentation/samples/?service=storage) biblioteka zawiera przykłady dla usługi Azure Storage, który można pobrać i uruchom lokalnie. Biblioteka przykładowy kod zawiera przykładowy kod w formacie zip. Alternatywnie można wybrać i klonowania repozytorium GitHub dla każdej próbki.

## <a name="net-samples"></a>Przykłady dla platformy .NET
Aby poznać przykłady .NET, Pobierz [biblioteki klienta usługi Storage .NET](https://www.nuget.org/packages/WindowsAzure.Storage/) z pakietu NuGet. Biblioteka klienta magazynu .NET jest również dostępna w [zestawu Azure SDK dla platformy .NET](https://azure.microsoft.com/downloads/).

* [Przykładów dla magazynu platformy Azure przy użyciu platformy .NET](storage-samples-dotnet.md)

## <a name="java-samples"></a>Przykłady w języku Java
Aby poznać przykłady Java, Pobierz [biblioteki klienta usługi Storage Java](https://github.com/azure/azure-storage-java).

* [Przykładów dla magazynu platformy Azure przy użyciu języka Java](storage-samples-java.md)

## <a name="nodejs-samples"></a>Przykłady dla platformy Node.js
Aby poznać przykłady Node.js, Pobierz [biblioteki klienta usługi Storage Node.js](https://github.com/Azure/azure-storage-node).

* [Moduł przesyłający obiektów blob](https://github.com/Azure/azure-storage-node/tree/master/examples/blobuploader)
* [Przekazywanie i pobieranie obiektu blob](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/blobuploaddownloadsample.js)
* [Token kontynuacji](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/continuationsample.js)
* [Zasady ponawiania](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/retrypolicysample.js)
* [Sygnatury dostępu współdzielonego](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/sassample.js)
* [Migawki](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/snapshotsample.js)
* [Zapytanie tabeli](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/tablequerysample.js)

## <a name="c-samples"></a>Przykłady dla języka C++
Aby poznać przykłady dla języka C++, Pobierz [biblioteki klienta usługi Storage C++](https://www.nuget.org/packages/wastorage/) z pakietu NuGet.

* [Rozpoczynanie pracy z obiektami blob](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/BlobsGettingStarted)
* [Rozpoczynanie pracy z tabelami](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/TablesGettingStarted)
* [Rozpoczynanie pracy z kolejki](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/QueuesGettingStarted)
