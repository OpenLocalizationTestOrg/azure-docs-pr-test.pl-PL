<span data-ttu-id="0f9f1-139">Informacje o aplikacji można rejestrować przy użyciu interfejsu API zestawu Mobile Engagement SDK lub przy użyciu interfejsu API urządzenia platformy Mobile Engagement.</span><span class="sxs-lookup"><span data-stu-id="0f9f1-139">App info can be registered by using the Mobile Engagement SDK API or by using the Mobile Engagement platform Device API.</span></span>

<span data-ttu-id="0f9f1-147">Dodatkowe dane (dodatki) to dowolne dane, które mogą być dołączone do zdarzeń, błędów, działań i zadań.</span><span class="sxs-lookup"><span data-stu-id="0f9f1-147">Extra data (or extras) is some arbitrary data that can be attached to events, errors, activities and jobs.</span></span>
