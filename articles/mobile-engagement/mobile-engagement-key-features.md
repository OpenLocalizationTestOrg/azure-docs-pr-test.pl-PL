# <a name="azure-mobile-engagement---key-features"></a>Azure Mobile Engagement — funkcje klucza
Ten artykuł zawiera omówienie wysokiego poziomu o najważniejsze funkcje platformy Mobile Engagement. 

## <a name="general"></a>**Ogólne**
* **Znaleźć zestawy SDK dla wszystkich głównych platform** dostępne dla wszystkich głównych platform - zestawy SDK dla systemu iOS, Android, uniwersalnych systemu Windows, Windows Phone Silverlight, urządzeń Kindle, Cordova. 
  Udostępniamy łatwo zintegrować zestawy SDK i przydatne dokumentacji ułatwiających rozpoczęcie pracy na dowolnej platformie wybranych przez użytkownika. 
* **Osobnego portalu na SaaS** umożliwia łatwy dostęp do zespołu marketingowego bez konieczności przejdź do portalu zarządzania Azure. 
* **Dostępność Otwórz interfejsów API REST** integracji i automatyzacji z systemów CRM/CMS/IT przy użyciu interfejsów API otwarty platformy, firma Microsoft udostępnia Otwórz interfejsów API REST i zestawu .NET SDK, aby korzystać z tych interfejsów API, które pozwalają można łatwo zintegrować i zautomatyzować z usługi Mobile Engagement. Zobacz [to](mobile-engagement-api-authentication.md) szczegółowe informacje. 
* **Łącznik Power BI dostępne** można również wysunąć wykresów analytics klucza do pulpitu nawigacyjnego usługi Power BI. Zobacz to [przewodnik](https://powerbi.microsoft.com/en-us/documentation/powerbi-content-pack-azure-mobile/)
* **Zapewnienie bezpieczeństwo i prywatność** usługi Azure Mobile Engagement jest częścią rodziny Azure wykonuje wszystkie standardowe najlepsze rozwiązania dotyczące zabezpieczeń i prywatności dla usługi w chmurze.
