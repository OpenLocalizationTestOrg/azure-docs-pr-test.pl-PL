# <a name="azure-mobile-engagement-web-sdk-release-notes"></a>Informacje o wersji zestaw SDK usługi Mobile Engagement w sieci Web Azure
## <a name="202-10182016"></a>2.0.2 (10/18/2016)
* Stałe awarii na InPrivate (Safari).
* Stałe awarii w przeglądarkach plików cookie jest wyłączona.

## <a name="201-6102016"></a>2.0.1 (6/10/2016)
* Wyłączone SDK w programie Internet Explorer 8 oraz Internet Explorer 9.
* Stałe wykrywania przeglądarki sieci web Opera.

## <a name="200-672016"></a>2.0.0 (6/7/2016)
* Wydane usługi Azure Mobile Engagement.
* Zaktualizowano, aby `appId` konfiguracji zastępuje konfigurację ciągu połączenia.
* Dodano interfejsy API pozwala włączać i wyłączać agenta.
* Ulepszenia zabezpieczeń.
* Zaktualizowano w celu użycia natywnego JSON interfejsów API z przeglądarki.
* Usunięte interfejsu API, która wysyła i odbiera komunikaty między urządzeniami.
* Usunięto związane z przestarzałe połączenia z protokołem XMPP wywołań zwrotnych.
* Usunięto obsługę funkcji Reach.
