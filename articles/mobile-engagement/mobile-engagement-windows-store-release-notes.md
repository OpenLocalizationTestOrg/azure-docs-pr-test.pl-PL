# <a name="windows-universal-apps-sdk-release-notes"></a>Informacje o wersji systemu Windows Universal SDK aplikacji
## <a name="341-11032016"></a>3.4.1 (11/03/2016)

* Ulepszenia.

## <a name="340-04192016"></a>3.4.0 (04/19/2016)
* Łączność ulepszenia nakładki.
* Dzienniki konsoli dodano "TestLogLevel" interfejsu API do filtru Włącz/Wyłącz emitowane przez zestaw SDK.
* Uruchom stały powiadomienia w działanie elementów docelowych pierwsze działanie, które nie są wyświetlane w aplikacji.

## <a name="331-02182016"></a>3.3.1 (02/18/2016)
* Stałe konfliktów między anonsu web zawartość HTML i strony HTML zestawu SDK.
* Ulepszenia.

## <a name="330-01222016"></a>3.3.0 (01/22/2016)
* Formatowanie w aplikacji platformy uniwersalnej systemu Windows działa w trybie wersji stałym awarii.
* Margines stałym 1px powiadomień dla aplikacji uniwersalnych 8.1.
* MS-appx ms appdata schematy i dostępne na adresy URL akcji.
* Ulepszenia.

## <a name="320-11202015"></a>3.2.0 (11/20/2015)
* Dodano obsługę aplikacji platformy uniwersalnej systemu Windows 10 systemu Windows.
* Kanał dodano wypychania udostępniania funkcji, aby rozwiązać konflikty kanał (teraz zgodne z usługą Azure Notification Hubs).
* Stałe awarii podczas żądania identyfikator urządzenia tylko po zainicjowaniu.
* Ulepszenia dzienniki konsoli.
* Stałe awarii podczas analizowania niektóre nieobsługiwane wyjątki.

## <a name="310-05212015"></a>3.1.0 (05/21/2015)
* Identyfikator urządzenia usługi Mobile Engagement teraz jest oparta na identyfikator GUID generowany podczas instalacji.

## <a name="301-04292015"></a>3.0.1 (04/29/2015)
* Stałe błędów wpływających na inicjowanie zestawu SDK na niektóre aplikacje Windows Phone WinRT.

## <a name="300-04032015"></a>3.0.0 (04/03/2015)
* Wprowadzenie do usługi Mobile Engagement SDK dla aplikacji uniwersalnej (z systemem Windows i Windows Phone WinRT).
* Domyślną ikonę powiadomienia aktualizacji.
* Wyślij opinię akcji powiadomień systemu Wstecz, po kliknięciu powiadomienie.
* Powiadomienie systemowe stały, która czasami jest odtwarzany w aplikacji po zostanie kliknięty.

## <a name="200-02172015"></a>2.0.0 (02/17/2015)
* Początkowa wersja usługi Azure Mobile Engagement
* Konfiguracja appId/sdkKey zastępuje konfigurację ciągu połączenia.
* Ulepszenia zabezpieczeń.
