# <a name="delete-a-user-from-a-directory-in-azure-active-directory"></a>Usuń użytkownika z katalogu w usłudze Azure Active Directory
W tym artykule opisano sposób toodelete użytkownika z katalogu w usłudze Azure Active Directory (Azure AD). Informacje dotyczące dodawania nowych użytkowników tooyour organizacji, zobacz [Dodaj nowe tooAzure użytkowników usługi Active Directory](active-directory-users-create-azure-portal.md).

## <a name="toodelete-a-user"></a>toodelete użytkownika
1. Zaloguj się za[hello portalu Azure](https://portal.azure.com) przy użyciu konta, które jest administratorem globalnym katalogu hello.
2. Wybierz **więcej usług**, wprowadź **użytkowników i grup** w hello pola tekstowego, a następnie wybierz **Enter**.

   ![Otwieranie Zarządzanie użytkownikami](./media/active-directory-users-delete-user-azure-portal/create-users-user-management.png)
3. Na powitania **użytkowników i grup** bloku, wybierz opcję **użytkowników**.

   ![Blok użytkownicy hello otwierania](./media/active-directory-users-delete-user-azure-portal/create-users-open-users-blade.png)
4. Na powitania **użytkowników i grup — Użytkownicy** bloku, wybierz użytkownika z listy hello.
5. W bloku hello hello wybranego użytkownika, wybierz **omówienie**, a następnie na pasku poleceń hello, wybierz **usunąć**.

    ![Wybranie polecenia Delete hello](./media/active-directory-users-delete-user-azure-portal/create-users-delete-command.png)
