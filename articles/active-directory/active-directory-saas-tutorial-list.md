# <a name="saas-application-integration-with-azure-active-directory"></a>Integracja aplikacji SaaS w usłudze Azure Active Directory

Aby zintegrować z włączoną obsługę chmury [(SaaS)](https://azure.microsoft.com/overview/what-is-saas/) aplikacji za pomocą usługi Azure Active Directory, opracowaliśmy kolekcji samouczków, które umożliwia przeprowadzenie konfiguracji.

Gdzie mamy wskazówki inicjowania obsługi administracyjnej użytkowników, firma Microsoft uwzględniła samouczek aplikacji użytkownika alokacji w prawej kolumnie obok samouczek aplikacji.

Aby uzyskać pełną listę aplikacji SaaS, które zostały wstępnie zintegrowanych z usługą Azure AD, zobacz [Active Directory Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps).
