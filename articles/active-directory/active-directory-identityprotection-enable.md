# <a name="enabling-azure-active-directory-identity-protection"></a>Włączenie ochrony tożsamości usługi Azure Active Directory
Azure Active Directory Identity Protection jest nową możliwość, która udostępnia skonsolidowany wgląd w podejrzane działania logowania i potencjalnych luk w zabezpieczeniach i przy użyciu powiadomień, zalecenia korygowania i ryzyka zasad pomaga chronić firmy. 

Usługi wykrywa podejrzane działania dla użytkowników końcowych i tożsamości uprzywilejowanych (Administrator) oparte na sygnały jak ataków siłowych, ujawnione poświadczenia logowania z nieznanych lokalizacji, zainfekowanych urządzeń, aby zapewnić ochronę przed te działania w czasie rzeczywistym. Co ważniejsze oparte na tych podejrzanych działań, ważność ryzyka użytkownika jest obliczana i ryzyka zasad można skonfigurować i automatycznie chronić tożsamości organizacji. Aby uzyskać więcej informacji, zobacz [Azure Active Directory Identity Protection](active-directory-identityprotection.md).

Tematy to przedstawiono sposób włączania usługi Azure Active Directory Identity Protection.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Kroki, aby włączyć usługi Azure Active Directory Identity Protection
1. [Zaloguj się na](https://ms.portal.azure.com/) do portalu Azure jako administrator globalny. 
2. W portalu Azure kliknij **Marketplace**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/01.png "tworzenie")
3. Na liście aplikacji kliknij **bezpieczeństwo i Obsługa tożsamości**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/02.png "tworzenie")
4. Kliknij przycisk **Azure AD Identity Protection**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/03.png "tworzenie")
5. Na **Azure AD Identity Protection** bloku, kliknij przycisk **Utwórz**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/04.png "tworzenie")
