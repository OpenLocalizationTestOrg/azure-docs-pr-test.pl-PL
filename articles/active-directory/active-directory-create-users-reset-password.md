# <a name="reset-a-password-for-a-user-in-the-azure-classic-portal"></a>Resetowanie hasła dla użytkownika w klasycznym portalu Azure
> [!div class="op_single_selector"]
> * [Witryna Azure Portal](active-directory-users-reset-password-azure-portal.md)
> * [Klasyczna witryna Azure Portal](active-directory-create-users-reset-password.md)
>
>

Azure Active Directory (Azure AD) upraszcza proces resetowania hasła użytkownika, Jeśli zapomnisz i są zablokowane, out lub podczas wykonywania zabezpieczeń rutynowej konserwacji. 

> [!IMPORTANT]
> Firma Microsoft zaleca zarządzanie usługą Azure AD przy użyciu [centrum administracyjnego usługi Azure AD](https://aad.portal.azure.com) w witrynie Azure Portal zamiast korzystania z klasycznej witryny Azure Portal przywołanej w niniejszym artykule. Jak zresetować hasło użytkownika w Centrum administracyjnym usługi Azure AD, zobacz [resetowania hasła dla użytkownika w usłudze Azure Active Directory](active-directory-users-reset-password-azure-portal.md).

1. Otwórz katalog.
2. Wybierz kartę **Użytkownicy**, a następnie wybierz nazwę wyświetlaną użytkownika, którą chcesz zmienić.
3. Na pasku poleceń Wybierz **Resetuj hasło**.
4. W oknie dialogowym resetowania hasła, kliknij przycisk **zresetować**.
5. Wybierz znacznik wyboru, aby zakończyć resetowanie hasła.
