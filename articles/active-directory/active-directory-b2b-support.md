# <a name="getting-support-for-b2b-collaboration"></a>Uzyskiwanie pomocy technicznej dla współpracy B2B

Zapoznaniu dokumentacji, to prawo rzeczy, ale nadal nie można pobrać elementu pracy? Otwórz bilet pomocy technicznej (wymagany jest plan pomocy technicznej):

1. W [portalu Azure](https://portal.azure.com), przejdź do bloku Pomoc i wsparcie techniczne i wybierz nowe żądanie pomocy technicznej:
  - Wystawianie typu: techniczne
  - Subskrypcję: Wybierz wpływ subskrypcji
    - Usługa: Usługi Active Directory
    - Plan pomocy technicznej: Plan pomocy technicznej odpowiednich wybierz

  ![nowe żądanie pomocy technicznej](media/active-directory-b2b-support/new-support-request.png)

2. Opisz problem:
  - Wybierz odpowiednie ważność, która odzwierciedla użytkownik potrzebuje.
  - Wybierz typ problemu jako użytkowników i grupy zarządzania
  - Wybierz kategorię jako Dodawanie użytkowników (B2B)
  - Zawierać komunikaty o błędach, takich jak CorrelationID, użytkowników i tak dalej.

  ![obsługuje biletu opis problemu](media/active-directory-b2b-support/problem-description.png)

3. Dodaj swoje informacje kontaktowe dla przedstawicielem pomocy technicznej w celu skontaktowania się z Tobą do dalszego rozwiązywania problemów.
