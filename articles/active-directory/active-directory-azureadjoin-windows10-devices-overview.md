# <a name="windows-10-for-the-enterprise-ways-to-use-devices-for-work"></a>System Windows 10 dla przedsiębiorstw: sposoby używania urządzenia do pracy
Windows 10 daje możliwość wykorzystać usługę Azure Active Directory (Azure AD). Urządzeń z systemem Windows 10 można połączyć z usługą Azure AD, dzięki czemu można logowania się do systemu Windows przy użyciu konta usługi Azure AD lub przez dodanie ich identyfikatorów Azure do uzyskiwania dostępu do aplikacji biznesowych i zasobów.

![Usługa Azure Active Directory z chmurą systemu Windows](./media/active-directory-azureadjoin/windows10-overview.png)
