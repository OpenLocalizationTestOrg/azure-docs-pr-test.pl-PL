Niedawna można tylko utworzysz nową subskrypcję Microsoft Azure przy użyciu konta Microsoft (Windows Live ID). Azure teraz obsługuje przy użyciu jednej z następujących hello dwa konta toosign metody zapasową:

* **Konta Microsoft** (utworzone przez użytkownika do użytku osobistego) — umożliwiają dostęp tooall konsumentów produktów firmy Microsoft i usługami w chmurze, takich jak Outlook (usługi Hotmail), programu Messenger, OneDrive, MSN, Xbox LIVE lub usługi Office 365. Podczas tworzenia skrzynki pocztowej Outlook.com automatycznie tworzy konto Microsoft. Po utworzeniu konta Microsoft, może to być usług chmurowych firmy Microsoft związane z odbiorców tooaccess używane lub Microsoft Azure. [Dowiedz się więcej](http://www.microsoft.com/account/default.aspx)
* **Konta służbowe** (wystawiony przez administratora dla firm/uczelni Użyj) - dostęp tooall małych, średnich i dużych firm poziom usług chmurowych firmy Microsoft, takich jak Azure, Microsoft Intune lub usługi Office 365. Po utworzeniu konta tooone tych usług jako organizacja katalogowych w chmurze jest automatycznie udostępniane w usłudze Azure Active Directory toorepresent Twojej organizacji. [Dowiedz się więcej](active-directory-administer.md)
