* <span data-ttu-id="49cbd-111">**Ulepszone zabezpieczenia** — dane są automatycznie szyfrowane przed opuszczeniem urządzenie użytkownika systemu Windows 10 za pomocą usługi Azure Rights Management (Azure RMS) i danych pozostaje zaszyfrowany magazynowanych w chmurze.</span><span class="sxs-lookup"><span data-stu-id="49cbd-111">**Enhanced security** – Data is automatically encrypted before leaving the user’s Windows 10 device by using Azure Rights Management (Azure RMS), and data stays encrypted at rest in the cloud.</span></span> <span data-ttu-id="49cbd-112">Cała zawartość pozostaje zaszyfrowany magazynowanych w chmurze, z wyjątkiem przestrzeni nazw, takich jak ustawienia nazwy i nazwy aplikacji systemu Windows.</span><span class="sxs-lookup"><span data-stu-id="49cbd-112">All content stays encrypted at rest in the cloud, except for the namespaces, like settings names and Windows app names.</span></span>
