# <a name="why-is-my-azure-subscription-disabled-and-how-do-i-reactivate-it"></a>Dlaczego Moja subskrypcja platformy Azure jest wyłączona i jak go uaktywnić?
Może mieć Twojej subskrypcji platformy Azure, które zostały wyłączone, ponieważ osiągnięto limit wydatków, ma zaległe rachunku, osiągnęła limit karty kredytowej, lub ponieważ subskrypcja została anulowana przez administratora konta. Zobacz, jaki problem dotyczy to Ciebie i wykonaj kroki opisane w tym artykule można pobrać subskrypcji ponownej aktywacji.

## <a name="you-reached-your-spending-limit"></a>Osiągnięto limit wydatków
Użycie zmagazynowane miesięcznej kwoty uwzględnionych w subskrypcji, możemy wyłączyć usługi na pozostałą część miesiąca rozliczeniowego. Usługa jest wyłączona w celu zabezpieczenia przed przypadkowym są naliczane opłaty za użycie poza kwota uwzględnionych w subskrypcji. Możesz usunąć limit wydatków i [skonfigurować alerty dotyczące subskrypcji platformy Microsoft Azure billing](billing-set-up-alerts.md) ułatwiają monitorowanie działań i zarządzania nimi rozliczeń dla konta platformy Azure.

Aby usunąć limit wydatków,

1. Zaloguj się do [Centrum konta platformy Azure](https://account.windowsazure.com/Home/Index).
2. Kliknij przycisk **subskrypcje**.
3. Wybierz subskrypcję.
4. Kliknij przycisk **subskrypcja osiągnęła Limit wydatków i została wyłączona, aby uniknąć naliczania opłat**.

    Jeśli ta opcja nie jest widoczny, kliknij przycisk **Usuń limit wydatków** w subskrypcji **stan** obszaru.
5. Wybierz jedną z następujących opcji:

| Opcja | Efekt |
| --- | --- |
| Usuń trwale limit wydatków |Usuwa limit wydatków, dopóki nie można włączyć ją ponownie. |
| Usuń limit wydatków w bieżącym okresie rozliczeniowym |Usuwa limit wydatków, ale limit wydatków, uruchamia ponownie w następnym okresie rozliczeniowym. |

Jeśli subskrypcji bezpłatnej wersji próbnej i Usuń limit wydatków, subskrypcja konwertuje się na płatność za rzeczywiste użycie w końcu bezpłatnej wersji próbnej.

## <a name="your-bill-is-past-due"></a>Rachunek jest zaległa
Należy zwrócić zaległe saldo w całości. Zapoznaj się z artykułem [dlaczego użytkownik otrzymał powiadomienie, że Twoja subskrypcja platformy Azure ma wcześniejszego termin saldo?](billing-azure-subscription-past-due-balance.md#how-to-resolve-the-issue)

## <a name="the-bill-exceeds-your-credit-card-limit"></a>Zestawienie przekracza limit Twojej karty kredytowej
Aby rozwiązać ten problem, [przełączyć się do innej karty kredytowej](billing-how-to-change-credit-card.md). Lub jeśli są reprezentujący firmy, możesz [przełącznik, aby zapłacić za faktury](https://azure.microsoft.com/pricing/invoicing/).

## <a name="the-subscription-was-canceled-by-the-account-administrator"></a>Subskrypcja została anulowana przez administratora konta
Jeśli jesteś administratorem konta, ponownie z subskrypcji przy użyciu następujących kroków:

1. Zaloguj się do [Centrum konta platformy Azure](https://account.windowsazure.com/Home/Index).
2. Wybierz subskrypcję, anulowane.
3. Kliknij przycisk **ponownie uaktywnić**.

![Zrzut ekranu pokazujący ponownie uaktywnić łącza w okienku po prawej stronie.](./media/billing-how-to-cancel-azure-subscription/reactivate-sub.png)

Dla innych typów subskrypcji [się z pomocą techniczną](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) aby swoją subskrypcję ponownej aktywacji.

Jeśli nadal potrzebujesz pomocy, [się z pomocą techniczną](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) uzyskać szybkie rozwiązanie problemu.
