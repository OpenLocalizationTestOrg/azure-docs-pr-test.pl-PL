# <a name="azure-security-mvp-program"></a>Program MVP zabezpieczeń platformy Azure
Microsoft najważniejszymi pracownikami pomocy technicznej (MVP) są użytkownicy społeczności, którzy wykazały, że przykładowy zobowiązań ułatwienia innym pełne wykorzystanie ich obsługi technologii firmy Microsoft. Mają ich męczennicy, wykorzystaniu praktycznej wiedzy i wiedzy technicznej ze społecznością i z firmą Microsoft.

Microsoft Azure rozpoznaje teraz społeczności ekspertów z specjalistyczną wiedzę w zabezpieczeń platformy Azure. MVPs Microsoft można przyznać MVP platformie Microsoft Azure w obszarze wkład zabezpieczeń Azure.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Nie testów porównawczych dla częściowo staje się MVP, ponieważ zależy to od technologii nie istnieje i jest cyklu życia, część kryteriów zawiera wpływ nominacji wkładów fora online, takich jak Microsoft Answers, TechNet i MSDN; stron typu wiki oraz w Internecie; konferencji i grup użytkowników; podkasty, witryn sieci Web, blogów i mediów społecznościowych; i artykułów i książek.

Czy eksperta w zabezpieczeń platformy Azure? Czy wiesz, osoby, która jest? Następnie [wyznaczyć siebie lub innej osoby](https://mvp.microsoft.com/Nomination/nominate-an-mvp) zostać zabezpieczeń platformy Azure MVP już dziś!
