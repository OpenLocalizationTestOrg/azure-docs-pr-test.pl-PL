# <a name="azure-operational-security-checklist"></a>Lista kontrolna zabezpieczeń platformy Azure operacyjne
Wdrażanie aplikacji na platformie Azure jest szybkie, łatwe i ekonomiczne. Przed wdrożeniem aplikacji w chmurze w środowisku produkcyjnym warto mieć listę kontrolną pomoc w ocenie aplikacja z listą bezpieczeństwa operacyjnego ważne i zalecane akcje należy wziąć pod uwagę.

## <a name="introduction"></a>Wprowadzenie

Platforma Azure udostępnia mechanizm usługi infrastruktury, które można użyć do wdrożenia aplikacji. Azure bezpieczeństwa operacyjnego odwołuje się do usług, formanty i funkcje dostępne dla użytkowników do ochrony danych, aplikacji i innych zasobów na platformie Microsoft Azure.

-   Aby uzyskać maksymalną wykorzystać możliwości platformy w chmurze, zaleca się korzystać z usług Azure i wykonaj listę kontrolną.
-   Organizacje, które inwestują czas i zasoby oceny gotowości operacyjnej aplikacjach przed uruchomieniem mają znacznie wyższym zadowolenia niż te, które nie. Podczas wykonywania tej pracy, listy kontrolne mogą być cenne mechanizm upewnij się, że aplikacje są oceniane spójnie i całościowo.
-   Poziom oceny operacyjne będą różni się w zależności od poziomu dojrzałości chmurze organizacji i etapie projektowania aplikacji, dostępności potrzeb i wymagań wrażliwości danych.

## <a name="conclusion"></a>Podsumowanie
Wiele organizacji zostały pomyślnie wdrożone i obsługiwane swoje aplikacje w chmurze na platformie Azure. Z list kontrolnych zawartych zaznacz kilka list kontrolnych, które są niezbędne i pomóc zwiększyć prawdopodobieństwo pomyślnego wdrożenia i zwolnij frustracji spowodowanej operacji. Zdecydowanie zaleca się powyższe zagadnienia operacyjnych i strategicznych dla istniejących i nowych wdrożeń aplikacji na platformie Azure.
