# <a name="get-started-with-hello-box-connector"></a>Rozpoczynanie pracy z hello pole łącznika
Łączenie tooBox i tworzenie plików, Usuń pliki i inne. Pole można:

* Tworzenie sieci przepływu biznesowe na podstawie danych hello, uzyskasz z pola. 
* Gdy plik jest tworzony lub aktualizowany za pomocą wyzwalaczy.
* Użyj akcji, które kopiowania pliku, usuń plik i inne. Te akcje odpowiedzi, a następnie wprowadź dane wyjściowe hello dostępne dla innych działań. Na przykład gdy plik zostanie zmieniona na pole, można wykonać tego pliku i poczty e-mail za pomocą usługi Office 365.

Rozpoczynanie pracy przez teraz tworzenie aplikacji logiki, zobacz [tworzenie aplikacji logiki](../logic-apps/logic-apps-create-a-logic-app.md).
