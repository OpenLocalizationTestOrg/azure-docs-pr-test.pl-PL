# <a name="get-started-with-the-excel-connector"></a>Rozpoczynanie pracy z łącznikiem programu Excel
Brak żadnego łącznika programu Excel w aplikacjach logiki. 

## <a name="to-use-excel-data"></a>Używanie danych programu Excel
Można przechowywać dane programu Excel jako plik wartości rozdzielanych przecinkami (CSV) w folderze pamięci masowej, takich jak [OneDrive](connectors-create-api-onedrive.md). Można również użyć tego pliku CSV z [łącznika płaski pliku](../logic-apps/logic-apps-enterprise-integration-flatfile.md).

## <a name="connector-specific-details"></a>Szczegóły dotyczące łącznika

Wyświetl wszystkie wyzwalacze i akcje zdefiniowane w swagger i zobacz też żadnych limitów w [szczegóły łącznika](/connectors/excel/).

## <a name="more-connectors"></a>Więcej łączników
Wróć do [listy interfejsów API](apis-list.md).
