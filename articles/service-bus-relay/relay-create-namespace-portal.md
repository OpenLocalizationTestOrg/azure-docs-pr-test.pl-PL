# <a name="create-a-relay-namespace-using-hello-azure-portal"></a>Tworzenie przestrzeni nazw przekazywania przy użyciu hello portalu Azure
Przestrzeń nazw jest wspólnym kontenerem dla wszystkich składników usługi Azure Relay. W jednej przestrzeni nazw może znajdować się wiele usług Relay, a przestrzenie nazw często pełnią rolę kontenerów aplikacji. Ma obecnie dwa różne sposoby toocreate przekazywania przestrzeni nazw.

1. Azure Portal (w tym artykule)
2. Szablony usługi [Azure Resource Manager](../azure-resource-manager/resource-group-overview.md)
