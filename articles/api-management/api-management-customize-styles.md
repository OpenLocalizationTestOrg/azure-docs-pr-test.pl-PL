<span data-ttu-id="db83a-109">Kolory, czcionki, rozmiary, odstępy i inne elementy stylu związane z dowolną stroną portalu są definiowane przez reguły stylów.</span><span class="sxs-lookup"><span data-stu-id="db83a-109">The colors, fonts, sizes, spacings, and other style-related elements of any page on the portal are defined by style rules.</span></span>

<span data-ttu-id="db83a-113">Następnie kliknij polecenie **Portal deweloperów** w prawym górnym rogu.</span><span class="sxs-lookup"><span data-stu-id="db83a-113">Then click on **Developer portal** on the top-right.</span></span>

<span data-ttu-id="db83a-115">Aby otworzyć pasek narzędzi dostosowywania, umieść wskaźnik myszy na ikonie dostosowywania (lub wybierz ją), a następnie kliknij pozycję „style” na pasku narzędzi.</span><span class="sxs-lookup"><span data-stu-id="db83a-115">To open the customization toolbar move your mouse over the customization icon (or select it) and then click "styles" from the toolbar.</span></span>

<span data-ttu-id="db83a-117">Istnieją dwa główne sposób edycji reguł stylów — można przeglądać listę wszystkich reguł stylów używanych w dowolnym miejscu (która jest wyświetlana domyślnie) i modyfikować styl zgodnie z potrzebami lub można wybrać pozycję **Wybierz element na stronie**, a następnie kliknąć dowolne miejsce na stronie, aby wyświetlić tylko style odpowiednie dla tego elementu.</span><span class="sxs-lookup"><span data-stu-id="db83a-117">There are two main way of editing styling rules - you can look through the list of all the style rules used anywhere which is displayed by default and modify a style as needed, or you can choose **Select an element on the page** and then click anywhere on the page to see only the styles for that element.</span></span>

![Pasek narzędzi do dostosowywania][api-management-customization-toolbar]

<span data-ttu-id="db83a-119">Na potrzeby tego przykładu kliknij opcję **Wybierz element na stronie**.</span><span class="sxs-lookup"><span data-stu-id="db83a-119">Click the **Select an element on the page** option for this example.</span></span>  <span data-ttu-id="db83a-120">Teraz umieszczenie kursora myszy nad elementami powoduje ich wyróżnienie, a kliknięcie wyróżnionego elementu pozwala rozpocząć edycję jego stylów.</span><span class="sxs-lookup"><span data-stu-id="db83a-120">Elements now become highlighted as you hover over them with the mouse to signify what element's styles you would start editing if you clicked.</span></span> <span data-ttu-id="db83a-121">Przenieś wskaźnik myszy na tekst w nagłówku (zazwyczaj jest tam nazwa firmy), a następnie kliknij go.</span><span class="sxs-lookup"><span data-stu-id="db83a-121">Move the mouse over the text in the header (typically you have the company name here), and then click it.</span></span> <span data-ttu-id="db83a-122">W edytorze stylów pojawi się zestaw nazwanych i podzielonych na kategorie reguł stylów.</span><span class="sxs-lookup"><span data-stu-id="db83a-122">A set of named and categorized styling rules appears within the styling editor.</span></span> <span data-ttu-id="db83a-123">Każda reguła reprezentuje właściwość stylu wybranego elementu.</span><span class="sxs-lookup"><span data-stu-id="db83a-123">Each rule represents a styling property of the selected element.</span></span> <span data-ttu-id="db83a-124">Na przykład w wybranym powyżej tekście nagłówka rozmiar tekstu to @font-size-h1, a nazwa czcionki z alternatywami to @headings-font-family.</span><span class="sxs-lookup"><span data-stu-id="db83a-124">For example, for the header text selected above, the size of the text is in @font-size-h1 while the name of the font with alternatives is in @headings-font-family.</span></span>

> <span data-ttu-id="db83a-125">Jeśli znasz środowisko [bootstrap][bootstrap], zauważysz, że te reguły są w rzeczywistości [zmiennymi LESS][LESS variables] w motywie bootstrap używanym w portalu dla deweloperów.</span><span class="sxs-lookup"><span data-stu-id="db83a-125">If you're familiar with [bootstrap][bootstrap], these rules are in fact [LESS variables][LESS variables] within the bootstrap theme used by the developer portal.</span></span>

<span data-ttu-id="db83a-126">Zmienimy teraz kolor tekstu nagłówka.</span><span class="sxs-lookup"><span data-stu-id="db83a-126">Let's change the color of the heading text.</span></span> <span data-ttu-id="db83a-127">Zaznacz zawartość pola **@headings-color** i wpisz **#000000**.</span><span class="sxs-lookup"><span data-stu-id="db83a-127">Select the entry in the **@headings-color** field and type **#000000**.</span></span> <span data-ttu-id="db83a-128">Jest to kod szesnastkowy koloru czarnego.</span><span class="sxs-lookup"><span data-stu-id="db83a-128">This is the hex code for the color black.</span></span> <span data-ttu-id="db83a-129">Wtedy na końcu pola tekstowego pojawi się wskaźnik koloru w kształcie kwadratu.</span><span class="sxs-lookup"><span data-stu-id="db83a-129">As you do this, you see that a square color indicator appears at the end of the text box.</span></span> <span data-ttu-id="db83a-130">Po kliknięciu tego wskaźnika można wybrać kolor z selektora kolorów.</span><span class="sxs-lookup"><span data-stu-id="db83a-130">If you click this indicator, a color picker lets you to choose a color.</span></span>

![Selektor kolorów][api-management-customization-toolbar-color-picker]

<span data-ttu-id="db83a-132">Zmiany można podglądać w czasie rzeczywistym podczas ich wprowadzania, ale są one widoczne tylko dla administratorów.</span><span class="sxs-lookup"><span data-stu-id="db83a-132">Changes are previewed in real time as you make them, but are visible only to administrators.</span></span> <span data-ttu-id="db83a-133">Aby zmiany stały się widoczne dla wszystkich użytkowników, kliknij przycisk **Publikuj** znajdujący się w edytorze stylów i zatwierdź zmiany.</span><span class="sxs-lookup"><span data-stu-id="db83a-133">To make these changes visible to everyone, click the **Publish** button in the styling editor and confirm the changes.</span></span>
