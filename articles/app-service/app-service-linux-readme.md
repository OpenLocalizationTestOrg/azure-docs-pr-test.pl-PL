# <a name="web-app-on-linux-documentation"></a>Aplikacja sieci Web w dokumentacji systemu Linux
Sieci Web aplikacji w systemie Linux jest obecnie w publicznej wersji zapoznawczej i umożliwia klientom działania aplikacji sieci web na platformie systemu Linux. Umożliwia lepszą zgodność aplikacji dla niektórych typów aplikacji i ułatwia migrację istniejących aplikacji sieci web hostowanej na platformie systemu Linux w innym miejscu na usługi aplikacji Azure.

Klienci mogą tworzyć aplikacje sieci web w systemie Linux tylko wewnątrz Linux włączone planu usługi App Service i ten Plan usługi App Service może zawierać tylko aplikacje sieci web opartych na systemie Linux. 

> Publiczny ceny dla aplikacji sieci Web w systemie Linux jest 50% ceny i cena jest obliczana na podstawie halving liczbę godzin obliczeń wykorzystanych w ramach planu usługi aplikacji.
> 
> Nie umowy dotyczącej poziomu usług jest oferowana w okresie publicznej wersji zapoznawczej.
