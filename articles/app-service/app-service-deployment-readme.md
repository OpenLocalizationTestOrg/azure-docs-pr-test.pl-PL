# <a name="azure-app-service-deployment-overview"></a>Omówienie wdrażania usługi aplikacji Azure
Usługa aplikacji Azure zapewnia bogaty i toosupport tworzenia przepływów pracy wdrażania wydajny i elastyczny zestaw zintegrowanych funkcji. Wdrażanie aplikacji, można wykorzystać opcje, takie jak ciągła integracja i kontroli źródła lokalnego publikowania, WebDeploy oraz FTP. Hello zaleca się, że metoda wdrażania aplikacji w produkcji jest wymiany miejsca wdrożenia. Miejsca wdrożenia reprezentuje środowisk przemieszczania i integracji skojarzonych z aplikacjami produkcji. Można konfigurować miejsc wdrożenia i zastosować ruchu w sieci web do sprawdzania poprawności i ruchu może być zamieniona na żądanie dla wdrożenia tooproduction bez dół czasu i automatyczne zwiększanie gotowości. Witaj kroki wdrażania przepływu pracy można łatwo zautomatyzować za pomocą wersji produktów zarządzania, takich jak program Visual Studio Release Management. Jest to przydatne w przypadku koordynacji z innych rozwiązań zasobów (np. Magazyn danych), cyklu i replikacji w wielu jednostkach wdrożenia.
