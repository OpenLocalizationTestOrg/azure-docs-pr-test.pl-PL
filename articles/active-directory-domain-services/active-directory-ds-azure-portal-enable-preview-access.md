# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Włączanie obsługi podglądu do administrowania domeny zarządzanej przy użyciu portalu Azure

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Istniejącej domeny zarządzanej nie jest wyświetlany w środowisku portalu Azure (wersja zapoznawcza)
Masz istniejącą [usług domenowych Azure AD](active-directory-ds-overview.md) domeny zarządzanej, który został utworzony przy użyciu klasycznego portalu Azure. Po przejściu do portalu Azure, blok usług domenowych Azure AD (wersja zapoznawcza) nie ma tego istniejącej domeny zarządzanej.

![Istniejącej domeny zarządzanej nie jest wyświetlana w portalu Azure](./media/getting-started/missing-domain-services-instance.png)

Jest to spowodowane istniejącej domeny zarządzanej nie zostały poddane do nowego portalu Azure. Zamierzasz zgłosić firmie Microsoft w celu włączenia nowe środowisko portalu Azure (wersja zapoznawcza) dla istniejącej domeny zarządzanej.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Dołącz Podgląd Azure środowisko portalu dla usług domenowych Azure AD
Postępuj zgodnie z instrukcjami [skontaktuj się z nami](active-directory-ds-contact-us.md) strony oraz Napisz do nas. Mamy szczególną przyjemność Włącz podgląd portalu Azure do istniejącej domeny zarządzanej.

> [!NOTE]
> Nowe środowisko portalu platformy Azure dla usług domenowych Azure AD jest ogólnie dostępna (GA), wszystkich istniejących domen zarządzanych automatycznie można migrować. W tym momencie można zarządzać istniejącej domeny zarządzanej przy użyciu portalu Azure.
