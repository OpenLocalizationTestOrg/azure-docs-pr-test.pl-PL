# <a name="installing-the-azure-toolkit-for-intellij"></a>Instalowanie zestawu narzędzi platformy Azure dla IntelliJ
Zestaw narzędzi platformy Azure dla IntelliJ zawiera szablony i funkcje, które umożliwiają łatwe tworzenie, tworzenie, testowanie i wdrażanie aplikacji platformy Azure przy użyciu środowiska programowania IntelliJ IDEA. Zestaw narzędzi platformy Azure dla IntelliJ jest projekt typu Open Source, którego kod źródłowy jest dostępny w ramach licencji MIT z projektu w witrynie GitHub pod adresem URL:

<https://github.com/Microsoft/Azure-Tools-for-Java>

Istnieją dwie metody instalowania zestawu narzędzi Azure for IntelliJ, w oknie dialogowym Ustawienia i z menu konfigurowanie na ekranie startowym; Obie metody instalacji będzie przedstawiono w poniższych krokach.

[!INCLUDE [azure-toolkit-for-IntelliJ-prerequisites](../includes/azure-toolkit-for-intellij-prerequisites.md)]

## <a name="to-install-the-azure-toolkit-for-intellij-from-the-settings-dialog-box"></a>Aby zainstalować zestaw narzędzi platformy Azure dla IntelliJ w oknie dialogowym Ustawienia
1. Uruchom środowisko IntelliJ IDEA.
2. Po otwarciu IntelliJ IDEA, kliknij **pliku**, następnie kliknij przycisk **ustawienia**.
   
    ![Otwórz okno dialogowe Ustawienia IntelliJ IDEA][01a]
3. W oknie dialogowym Ustawienia, kliknij przycisk **wtyczek**, a następnie kliknij przycisk **Przeglądaj repozytoria**.
   
    ![Okno dialogowe Ustawienia ROZWIĄZANIEM IntelliJ][02a]
4. W **Przeglądaj repozytoria** oknie dialogowym wpisz "Azure" w polu wyszukiwania. Wyróżnij **narzędzi Azure dla IntelliJ**, a następnie kliknij przycisk **zainstalować**.
   
    ![Wyszukiwanie Azure Toolkit IntelliJ][03]
   
    IntelliJ IDEA wyświetli się postęp instalacji w oknie dialogowym.
   
    ![Postęp instalacji][04]
5. Po zakończeniu instalacji kliknij przycisk **ponowne uruchomienie IntelliJ IDEA**.
   
    ![Uruchom ponownie IntelliJ IDEA][05]
6. Kliknij przycisk **OK** aby zamknąć okno dialogowe Ustawienia.
   
    ![Zamknij okno dialogowe Ustawienia ROZWIĄZANIEM IntelliJ][06]
7. Po wyświetleniu monitu o ponowne uruchomienie IntelliJ IDEA lub odłożyć, kliknij przycisk **ponownego uruchomienia**.
   
    ![Uruchom ponownie IntelliJ IDEA][07]

## <a name="to-install-the-azure-toolkit-for-intellij-from-the-start-screen"></a>Aby zainstalować zestaw narzędzi platformy Azure dla IntelliJ z ekranu startowego
1. Uruchom środowisko IntelliJ IDEA.
2. Po wyświetleniu ekranu startowego IntelliJ IDEA kliknij **Konfiguruj**, następnie kliknij przycisk **wtyczek**.
   
    ![Zainstaluj środowisko IntelliJ IDEA wtyczek][01b]
3. W **wtyczek** okno dialogowe, kliknij przycisk **Przeglądaj repozytoria**.
   
    ![Przeglądaj IntelliJ IDEA wtyczki repozytoria][02b]
4. W **Przeglądaj repozytoria** oknie dialogowym wpisz "Azure" w polu wyszukiwania. Wyróżnij **narzędzi Azure dla IntelliJ**, a następnie kliknij przycisk **zainstalować**.
   
    ![Wyszukiwanie Azure Toolkit IntelliJ][03]
   
    IntelliJ IDEA wyświetli się postęp instalacji w oknie dialogowym.
   
    ![Postęp instalacji][04]
5. Po zakończeniu instalacji kliknij przycisk **ponowne uruchomienie IntelliJ IDEA**.
   
    ![Uruchom ponownie IntelliJ IDEA][05]
6. Po wyświetleniu monitu o ponowne uruchomienie IntelliJ IDEA lub odłożyć, kliknij przycisk **ponownego uruchomienia**.
   
    ![Uruchom ponownie IntelliJ IDEA][07]
