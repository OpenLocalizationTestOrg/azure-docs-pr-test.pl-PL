# <a name="microsoft-azure-new-support-request-file-upload-utility-guidelines"></a>Przekazywanie pliku wskazówki dotyczące narzędzia żądania obsługi nowych Microsoft Azure
Gdy użytkownik [Utwórz żądanie obsługi](https://portal.azure.com/#create/Microsoft.Support), narzędzie przekazywanie pliku pomoże Ci łatwo przekazywania plików diagnostycznych lub innych plików, które prawdopodobnie są istotne zdarzenia.  

> [!NOTE]
> Aby chronić Twoją prywatność, nie ma żadnych danych osobowych w przekazania.
>
>

Za pomocą narzędzia przekazywanie pliku, należy przestrzegać następujących wytycznych:

* Wszystkie pliki muszą mieć rozszerzenie nazwy pliku, np. docx lub xlsx. Poniższa tabela listy dozwolonych rozszerzeń nazw.
* Nazwa pliku może być dłuższa niż 110 znaków.
* Nie można przekazać więcej niż jeden plik.
* Plików nie może być większy niż 4 MB.
