# <a name="azure-toolkit-for-eclipse"></a>Zestaw narzędzi platformy Azure dla środowiska Eclipse
Zestaw narzędzi platformy Azure dla programu Eclipse zawiera szablony i funkcje, które można łatwo utworzyć, rozwijać, testowania i wdrażania aplikacji Azure za pomocą programu Eclipse zintegrowane środowisko programistyczne (IDE). Jest to projekt open source, którego kod źródłowy jest dostępny w obszarze [MIT licencji z witryny projektu](https://github.com/microsoft/azure-tools-for-java).

> [!NOTE]
> Dostępna jest również zestaw narzędzi platformy Azure dla IntelliJ IDE. Aby uzyskać więcej informacji, zobacz [narzędzi Azure dla IntelliJ](azure-toolkit-for-intellij.md).
