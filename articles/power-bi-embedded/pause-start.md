# <a name="pause-and-start-your-power-bi-embedded-capacity-in-the-azure-portal"></a>Wstrzymywanie i uruchomić Power BI Embedded wydajność w portalu Azure

W tym artykule przedstawiono sposób wstrzymywania i uruchomić Power BI Embedded pojemności w Microsoft Azure. Przyjęto założenie, że utworzono Power BI Embedded pojemności. Jeśli nie masz, zobacz [utworzyć Power BI Embedded pojemności w portalu Azure](create-capacity.md) rozpocząć pracę.

Jeśli nie masz subskrypcji platformy Azure, przed rozpoczęciem utwórz [bezpłatne konto](https://azure.microsoft.com/free/).

## <a name="pause-your-capacity"></a>Wstrzymywać wydajność

Wstrzymanie możliwości uniemożliwia są rozliczane. Wstrzymywanie wydajność jest doskonałym rozwiązaniem, jeśli jest konieczne pojemność w danym okresie czasu. Wykonaj następujące kroki, aby wstrzymać wydajność.

> [!NOTE]
> Wstrzymywanie pojemności może uniemożliwić zawartości są dostępne w ramach usługi Power BI. Upewnij się, że przypisanie obszarów roboczych z możliwości przed wstrzymaniem aby zapobiegać przerwom w.

1. Zaloguj się do [Azure Portal](https://portal.azure.com/).

2. Wybierz **więcej usług** > **Power BI Embedded** wyświetlić pojemności sieci.

    ![Więcej usług w portalu Azure](media/pause-start/azure-portal-more-services.png)

3. Wybierz pojemności, którą chcesz wstrzymać.

    ![Power BI Embedded listy pojemności w portalu Azure](media/pause-start/azure-portal-capacity-list.png)

4. Wybierz **Wstrzymaj** w ramach szczegółów pojemności.

    ![Wstrzymywać wydajność](media/pause-start/azure-portal-pause-capacity.png)

5. Wybierz **tak**, który potwierdza, że chcesz wstrzymać pojemność.

    ![Upewnij się, Wstrzymaj](media/pause-start/azure-portal-confirm-pause.png)

## <a name="start-your-capacity"></a>Uruchom wydajność

Wznów użycia przez uruchomienie wydajność. Uruchamianie wydajność również wznawia rozliczeń.

1. Zaloguj się do [Azure Portal](https://portal.azure.com/).

2. Wybierz **więcej usług** > **Power BI Embedded** wyświetlić pojemności sieci.

    ![Więcej usług w portalu Azure](media/pause-start/azure-portal-more-services.png)

3. Wybierz pojemności, który chcesz uruchomić.

    ![Power BI Embedded listy pojemności w portalu Azure](media/pause-start/azure-portal-capacity-list.png)

4. Wybierz **Start** w ramach szczegółów pojemności.

    ![Uruchom wydajność](media/pause-start/azure-portal-start-capacity.png)

5. Wybierz **tak**, który potwierdza, że chcesz uruchomić pojemność.

    ![Potwierdź uruchomienie](media/pause-start/azure-portal-confirm-start.png)

Jeśli zawartość jest przypisany do tej pojemności, jest ona dostępna po uruchomieniu.
