<span data-ttu-id="fd994-467">Wirtualne centrum danych jest podejściu do migracji centrum danych w chmurze, która używa kombinacji funkcji i możliwości w celu utworzenia Skalowalna architektura na platformie Azure, które pozwala zmaksymalizować wykorzystanie zasobów chmury, zmniejszenie kosztów i uproszczenia zarządzania systemu.</span><span class="sxs-lookup"><span data-stu-id="fd994-467">The Virtual Data Center is an approach to data center migration into the cloud that uses a combination of features and capabilities to create a scalable architecture in Azure that maximizes cloud resource use, reducing costs, and simplifying system governance.</span></span> <span data-ttu-id="fd994-468">Pojęcie vDC opiera się na topologii partnerzy koncentratora, udostępnia typowe usługi udostępnionych w Centrum, dzięki czemu określonych aplikacji/obciążeń w klienci.</span><span class="sxs-lookup"><span data-stu-id="fd994-468">The vDC concept is based on a hub-spokes topology, providing common shared services in the hub and allowing specific applications/workloads in the spokes.</span></span> <span data-ttu-id="fd994-469">VDC odpowiada struktury ról firmy, gdzie różnych działów (centralnym dziale IT, metodyki DevOps, operacji i konserwacji) współdziałają ze sobą, każda z określonej listy ról i obowiązków.</span><span class="sxs-lookup"><span data-stu-id="fd994-469">A vDC matches the structure of company roles, where different departments (Central IT, DevOps, operation and maintenance) work together, each with a specific list of roles and duties.</span></span> <span data-ttu-id="fd994-470">VDC spełnia wymagania dotyczące migracji "Przyrostu i Shift", ale również zapewnia wiele zalet wdrożenia natywnego chmury.</span><span class="sxs-lookup"><span data-stu-id="fd994-470">A vDC satisfies the requirements for a "Lift and Shift" migration, but also provides many advantages to native cloud deployments.</span></span>
