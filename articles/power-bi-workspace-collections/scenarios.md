# <a name="common-power-bi-workspace-collection-scenarios"></a>Typowe scenariusze kolekcji obszarów roboczych usługi Power BI

Power BI obszaru roboczego kolekcje koncentruje się na udostępnianie większość istniejące funkcje interfejsu API programu Power BI jako część usługi Azure umożliwiające opracowanie aplikacji przy użyciu.  Ponadto jest możliwość programowo obsługi administracyjnej, opracowywania i wdrażania zasobów konieczne i zawartość usługi Power BI. Aby dowiedzieć się więcej, zobacz [wprowadzenie do programu Microsoft Power BI obszaru roboczego kolekcje](get-started.md).

> [!IMPORTANT]
> Kolekcje obszarów roboczych usługi Power BI są przestarzałe i będą dostępne do czerwca 2018 roku lub do daty podanej w kontrakcie. Zachęcamy do zaplanowania migracji do usługi Power BI Embedded, aby uniknąć przerw w działaniu aplikacji. Aby uzyskać informacje dotyczące sposobu przeprowadzenia migracji danych do usługi Power BI Embedded, zobacz [How to migrate Power BI Workspace Collections content to Power BI Embedded (Migrowanie zawartości kolekcji obszarów roboczych usługi Power BI do usługi Power BI Embedded)](https://powerbi.microsoft.com/documentation/powerbi-developer-migrate-from-powerbi-embedded/).

Poniżej przedstawiono kilka scenariuszy stosowania **Power BI obszaru roboczego kolekcje** w aplikacji.

* Pełni interakcyjnych raportów, które zostały utworzone w programie Power BI Desktop można ją osadzić w aplikacji przy użyciu nowego modelu token uwierzytelniania aplikacji.
* Niezależni dostawcy oprogramowania i klienci kompilujący klientów aplikacje mogą korzystać z **kolekcji obszarów roboczych usługi Power BI** usługi i Power BI zestaw SDK, aby osadzić interakcyjnych raportów. Aby dowiedzieć się więcej, zobacz temat [wprowadzenie do programu Microsoft Power BI obszaru roboczego kolekcje](get-started.md).
* Deweloperzy framework wizualizacji usługi Power BI służy do tworzenia niestandardowych wizualizacje, które mogą być używane we własnych aplikacjach. Zobacz [rozszerzenie usługi Power BI z niestandardowe elementy wizualne](https://powerbi.microsoft.com/custom-visuals/).
