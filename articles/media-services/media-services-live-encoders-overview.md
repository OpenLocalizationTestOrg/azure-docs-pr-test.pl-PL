# <a name="how-tooconfigure-on-premises-encoders-when-using-azure-media-services-toocreate-multi-bitrate-streams"></a>Jak tooconfigure lokalnymi koderów podczas korzystania z usługi Azure Media Services toocreate wielokrotnej szybkości transmisji bitów strumieni
W tym temacie wymieniono lokalnymi koderów na żywo można użyć toocapture wydarzeń na żywo i wysyłać strumień na żywo tooAMS kanały (włączyć kodowanie na żywo) dla dalszego przetwarzania pojedynczej szybkości transmisji bitów. Witaj temat zawiera także linki tootutorials, które pokazują, jak tooconfigure wymienione koderów.

## <a name="flash-media-live-encoder"></a>Koder na żywo Flash nośnika

## <a name="telestream-wirecast"></a>Telestream Wirecast
