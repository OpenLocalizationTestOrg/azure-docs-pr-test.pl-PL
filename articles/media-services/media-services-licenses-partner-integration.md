# <a name="using-partners-toodeliver-widevine-licenses-tooazure-media-services"></a>Przy użyciu partnerów toodeliver Widevine licencji tooAzure Media Services
## <a name="overview"></a>Omówienie
Microsoft Azure Media Services umożliwia toodeliver możesz MPEG-DASH chroniony z Widevine DRM, są szyfrowane na powitania specyfikacji wspólnego szyfrowania (CENC).

Począwszy od hello SDK .NET usługi Media Services wersja 3.5.2 Media Services umożliwia możesz tooconfigure szablonu licencji Widevine i uzyskać licencji Widevine. Można również użyć powitania po dostarczania licencji Widevine toohelp partnerów AMS: [Axinom](http://www.axinom.com/press/ibc-axinom-drm-6/), [EZDRM](http://ezdrm.com/), [castLabs](http://castlabs.com/company/partners/azure/).

## <a name="castlabs"></a>castLabs
Można użyć [castLabs](http://castlabs.com/company/partners/azure/) toodeliver licencji Widevine. Aby uzyskać więcej informacji, zobacz [przy użyciu castLabs toodeliver DRM licencje tooAzure Media Services](media-services-castlabs-integration.md)

## <a name="axinom"></a>Axinom
Można użyć [Axinom](http://www.axinom.com/press/ibc-axinom-drm-6/) toodeliver licencji Widevine. Aby uzyskać więcej informacji, zobacz [toodeliver Axinom przy użyciu DRM licencje tooAzure Media Services](media-services-axinom-integration.md)
