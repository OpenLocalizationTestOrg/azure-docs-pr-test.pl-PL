# <a name="deploy-popular-application-frameworks-on-windows-using-azure-resource-manager-templates"></a>Wdrażanie popularnych struktur aplikacji w systemie Windows za pomocą szablonów usługi Azure Resource Manager 

Obciążenia zwykle wymagają wielu zasobów, które działają zgodnie z projektu. Szablony usługi Azure Resource Manager umożliwiają dla można nie tylko określają sposób konfiguracji aplikacji, ale także jak zasoby są wdrażane aplikacje skonfigurowane pomocy technicznej. W tym artykule przedstawiono najpopularniejsze szablony w galerii i zawiera informacje dotyczące korzystania z portalu Azure, Azure CLI lub programu PowerShell dotyczące wdrażania ich.
