# <a name="managing-azure-virtual-machines-using-azure-automation"></a>Zarządzanie usługą Azure Virtual Machines przy użyciu usługi Azure Automation
W tym przewodniku przedstawiono usługi Automatyzacja Azure i jak może służyć aby ułatwić zarządzanie maszynach wirtualnych platformy Azure.

## <a name="what-is-azure-automation"></a>Co to jest Azure Automation?
[Automatyzacja Azure](https://azure.microsoft.com/services/automation/) jest usługą platformy Azure, dla uproszczenia zarządzania chmurą za pomocą automatyzacji procesu. Za pomocą usługi Automatyzacja Azure, można zautomatyzować długotrwałe, ręcznych, podatnych i często powtarzanych zadań, aby zwiększyć niezawodność, wydajność i czasu na wartość dla organizacji.

Automatyzacja Azure umożliwia aparatowi wykonywania przepływów pracy wysoce niezawodne i wysokiej dostępności, która może obsłużyć do własnych potrzeb miarę rozwoju organizacji. W automatyzacji Azure procesów może być rozpoczęte ręcznie, przez systemy innych firm lub w zaplanowanych odstępach czasu tak, aby zadania się tak zdarzyć, dokładnie tak, gdy jest to wymagane.

Można zmniejszyć nakłady operacyjne i zwolnić IT i pracownicy DevOps skupić się na pracy dodającego wartość biznesową, uruchamiając chmury zadań zarządzania automatycznie przy użyciu automatyzacji Azure.

## <a name="how-can-azure-automation-help-manage-azure-virtual-machines"></a>Jak usługi Automatyzacja Azure ułatwia zarządzanie maszyn wirtualnych platformy Azure?
Maszyny wirtualne można zarządzać w automatyzacji Azure za pomocą [programu Azure PowerShell](https://msdn.microsoft.com/library/azure/jj156055.aspx). Automatyzacja Azure obejmuje poleceń cmdlet programu Azure PowerShell, można wykonywać wszystkie zadania zarządzania maszyny wirtualnej w ramach usługi. Można również skojarzyć poleceń cmdlet w automatyzacji Azure za pomocą poleceń cmdlet dla innych usług Azure, aby zautomatyzować złożone zadania między usługami platformy Azure i systemów innych firm.
