# <a name="create-an-office-data-connection-file"></a>Tworzenie pliku połączenia danych pakietu Office

Zaktualizowano [dostawcy MSOLAP.7](analysis-services-data-providers.md) jest również wymagany.


1. Skopiuj plik połączenia próbki hello poniżej i wkleić do edytora tekstu. 

2. W `odc:ConnectionString`, zmień hello następujące właściwości:

    *   W `Initial Catalog=<database>;` zmienić `<database>` toohello Nazwa bazy danych.

3. W `<odc:CommandText>Model</odc:CommandText>` zmienić `Model` toohello nazwę modelu lub perspektywy. 

4. Zapisz plik hello z `.odc` toohello rozszerzenia C:\Users\\*username*\Documents\My folder źródła danych.

5. Kliknij prawym przyciskiem myszy plik hello, a następnie kliknij przycisk **Otwórz w programie Excel**. Lub w programie Excel na powitania **danych** wstążki, kliknij przycisk **istniejących połączeń**, wybierz plik, a następnie kliknij przycisk **Otwórz**.
