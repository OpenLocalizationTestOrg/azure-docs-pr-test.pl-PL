# <a name="use-the-storsimple-device-manager-service-to-log-a-support-request-for-the-storsimple-virtual-array"></a>Za pomocą usługi Menedżera urządzeń StorSimple dziennika żądania pomocy technicznej dla tablicy wirtualnego StorSimple

## <a name="overview"></a>Omówienie

Menedżer urządzeń StorSimple oferuje możliwość **dziennika nowe żądanie pomocy technicznej** wewnątrz bloku podsumowania usługi. W tym artykule opisano sposób logowania nowe żądanie pomocy technicznej i zarządzać jego cyklem od w portalu.

## <a name="new-support-request"></a>Nowe żądanie pomocy technicznej

W zależności od Twojego [plan pomocy technicznej](https://azure.microsoft.com/support/plans/), tworzenia biletów pomocy technicznej dla problemu na tablica bezpośrednio z bloku podsumowania usługi Menedżer StorSimple urządzenia wirtualnego StorSimple.

#### <a name="to-log-a-new-request"></a>Nowe żądanie logowania

1. Przejdź do usługi Menedżer urządzeń StorSimple. W bloku Podsumowanie ustawień usługi, przejdź do **pomocy technicznej i rozwiązywania problemów** sekcji, a następnie kliknij przycisk **nowy obsługuje żądania**.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket1.png)

2. W **podstawy** blok, wykonaj następujące czynności:

    1. Z **wydawania typu** listy rozwijanej wybierz **techniczne**. 
    
    2. Bieżący **subskrypcji**, **usługi** typu i **zasobów** (usługa Menedżera urządzeń StorSimple) są automatycznie wybrana. 

    3. Określ co najmniej jeden urządzenia zarejestrowane w usłudze występują problemy.

    4. Wybierz odpowiednie **plan pomocy technicznej** Jeśli masz wiele planów skojarzonych z Twoją subskrypcją. Należy plan płatnej pomocy technicznej, aby włączyć pomocy technicznej.

3. W **krok 2**, wybierz **ważność** i określ, czy problem jest związana z tablicy lub usługi Menedżer StorSimple urządzenia. Możesz także określić **kategorii** w tym problemie i podaj więcej **szczegóły** o problemie.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket2.png)

4. W **kroku 3**, podaj swoje informacje kontaktowe. Microsoft Support użyje tych informacji do dotrzeć do Ciebie, aby uzyskać więcej informacji, diagnozowanie i rozwiązywanie.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket3.png)

## <a name="manage-a-support-request"></a>Zarządzanie żądania obsługi

Po utworzeniu biletu pomocy technicznej możesz zarządzać jego cyklem życia z poziomu portalu.

#### <a name="to-manage-your-support-requests"></a>Do zarządzania własnych żądań pomocy technicznej

Aby przejść do strony pomocy i obsługi technicznej, przejdź do **Przeglądaj > Pomoc i obsługa techniczna**.

![Zarządzanie żądaniami obsługi](./media/storsimple-virtual-array-log-support-ticket/manage-support-tickets.png)
