# <a name="azure-database-for-mysql-documentation"></a>Dokumentacja usługi Azure Database dla MySQL

Usługa Azure Database dla MySQL jest usługą relacyjnej bazy danych opartą na aparacie bazy danych MySQL Server typu open source.  Jest to w pełni zarządzana baza danych oferowana jako usługa, za pomocą której można obsługiwać obciążenia o krytycznym znaczeniu — z przewidywalną wydajnością i dynamiczną skalowalnością. Twórz aplikacje za pomocą usługi Azure Database dla MySQL, korzystając z narzędzi open source i wybranej platformy. Dowiedz się, jak używać usługi Azure Database dla MySQL, korzystając z przewodników Szybki Start, samouczków i przykładów.
