# <a name="azure-cosmos-db-documentation"></a>Dokumentacja usługi Azure Cosmos DB

Azure Cosmos DB to usługa rozproszonej globalnie bazy danych umożliwiająca elastyczne i niezależne skalowanie przepływności i magazynu w dowolnej liczbie regionów geograficznych z kompleksową umową SLA. Za pomocą usługi Cosmos DB można utworzyć dokument, klucz/wartość i grafowe bazy danych, korzystając z serii popularnych interfejsów API i modeli programowania. Dowiedz się, jak używać usługi Cosmos DB, korzystając z przewodników Szybki Start, samouczków i przykładów.
