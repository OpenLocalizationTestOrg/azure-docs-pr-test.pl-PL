## <a name="append-other-apis"></a>Dołącz innych interfejsów API

Interfejs API może składać się z interfejsów API udostępnianych przez różnych usług: specyfikacja OpenAPI, SOAP API, aplikacji interfejsu API, aplikacji funkcji aplikacji logiki, sieci szkieletowej usług.

![Import](./media/api-management-append-apis/import.png)

Aby dołączyć inny interfejs API do istniejącego interfejsu API, wykonaj poniższe kroki. Po zaimportowaniu innego interfejsu API, operacje są dołączane do bieżącego interfejsu API.

1. Przejdź do wystąpienia APIM w portalu Azure.
2. Wybierz interfejsy API z interfejsu API zarządzania.
3. Naciśnij przycisk wielokropka ". . ." tp dalej interfejsu API, który chcesz dołączyć innego interfejsu API.
4. Wybierz opcję Importuj z menu rozwijanego.
5. Wybierz jedną z usług, z których chcesz zaimportować interfejsu API.
