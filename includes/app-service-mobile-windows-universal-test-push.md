
1. Kliknij prawym przyciskiem myszy projekt Sklepu Windows hello, kliknij przycisk **Ustaw jako projekt startowy**, naciśnij klawisz hello F5 klucza toorun hello w Sklepie Windows.
   
    Po uruchomieniu aplikacji hello hello urządzenie jest zarejestrowane dla powiadomień wypychanych.
2. Zatrzymaj aplikację ze Sklepu Windows hello i powtórz krok powitania dla aplikacji ze Sklepu Windows Phone hello.
   
    W tym momencie zarówno urządzenia są zarejestrowane tooreceive powiadomień wypychanych.
3. Ponownie uruchom aplikację ze Sklepu Windows hello i wpisz tekst w **Wstaw czynność do wykonania**, a następnie kliknij przycisk **zapisać**.
   
       Note that after hello insert completes, both hello Windows Store and hello Windows Phone apps receive a push notification from WNS. hello notification is displayed on Windows Phone even when hello app isn't running.
