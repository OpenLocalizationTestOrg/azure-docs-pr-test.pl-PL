
1. Kliknij prawym przyciskiem myszy projekt Sklepu Windows, kliknij przycisk **Ustaw jako projekt startowy**, naciśnij klawisz F5, aby uruchomić aplikację ze Sklepu Windows.
   
    Po uruchomieniu aplikacji, urządzenie jest zarejestrowane dla powiadomień wypychanych.
2. Zatrzymaj aplikację ze Sklepu Windows i powtórz poprzedni krok dla aplikacji do Sklepu Windows Phone.
   
    W tym momencie zarówno na urządzeniach zarejestrowanych do odbierania powiadomień wypychanych.
3. Ponownie uruchom aplikację ze Sklepu Windows, a następnie wpisz tekst w **Wstaw czynność do wykonania**, a następnie kliknij przycisk **zapisać**.
   
       Note that after the insert completes, both the Windows Store and the Windows Phone apps receive a push notification from WNS. The notification is displayed on Windows Phone even when the app isn't running.
