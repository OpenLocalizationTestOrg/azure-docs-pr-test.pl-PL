W razie potrzeby Zainstaluj moduł Azure PowerShell, korzystając z instrukcji w [Przewodnik programu Azure PowerShell](https://docs.microsoft.com/powershell/azureps-cmdlets-docs/), a następnie uruchom `Login-AzureRmAccount` można utworzyć połączenia z platformą Azure. Ponadto musisz mieć klucz publiczny SSH o nazwie `id_rsa.pub` w katalogu .ssh profilu użytkownika.
